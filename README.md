## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To create a production build:

```bash
npm run build
# or
yarn build
```

To run the production build locally:

```bash
npm run start
# or
yarn start
```
